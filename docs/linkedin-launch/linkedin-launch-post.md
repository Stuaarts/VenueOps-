# LinkedIn Launch Post

I just published VenueOps, a full-stack event operations platform built to reflect real venue and hospitality workflows.

Live demo: https://venueops-web.onrender.com  
GitHub: https://github.com/Stuaarts/VenueOps-

VenueOps manages:

- Event bookings and booking status changes
- Venue rooms, client records, and guest counts
- Staff assignment workflows
- Shift notes and operational handoff details
- Role-based dashboards for managers, staff, admins, and demo users

Tech stack:

- ASP.NET Core 8 Web API
- React, TypeScript, and Vite
- PostgreSQL with Entity Framework Core migrations
- JWT authentication and role-based access control
- Docker Compose for local development
- GitHub Actions CI for backend, frontend, and Docker validation
- Free Render deployment for the public demo

This project was built as a portfolio piece to show more than CRUD. I wanted it to feel like software a real operations team could use before, during, and after events.

The repo includes screenshots, a demo video, architecture notes, deployment docs, seeded demo credentials, tests, Docker setup, and CI.
