# Resume Bullets

- Built VenueOps, a full-stack event operations platform using ASP.NET Core, React, TypeScript, PostgreSQL, Docker, and GitHub Actions to manage venue bookings, staff assignments, guest counts, and shift notes.
- Designed REST APIs, Entity Framework Core data models, validation rules, seeded demo data, and role-based authorization for realistic hospitality operations workflows.
- Implemented a responsive React dashboard with booking filters, status updates, staff assignment forms, shift note entry, and role-aware navigation for Admin, Manager, Staff, and Demo users.
- Containerized the backend, frontend, and PostgreSQL database with Docker Compose and added CI checks for backend tests, frontend lint/test/build, and Docker image builds.
- Deployed a public portfolio demo on Render with fake demo credentials, startup migrations, seeded records, and documented setup, architecture, screenshots, and deployment tradeoffs.
