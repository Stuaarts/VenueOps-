# LinkedIn Project Entry

## VenueOps

Full-stack event operations platform for managing venue bookings, staff assignments, guest counts, and shift notes.

Live demo: https://venueops-web.onrender.com  
GitHub: https://github.com/Stuaarts/VenueOps-

## Description

Built a production-style portfolio application inspired by real hospitality and event operations workflows. VenueOps includes role-based dashboards, JWT authentication, booking CRUD, staff assignment management, shift notes, PostgreSQL persistence, Docker Compose local setup, and GitHub Actions CI.

## Technologies

ASP.NET Core 8, C#, React, TypeScript, Vite, PostgreSQL, Entity Framework Core, Docker, GitHub Actions, Render

## Highlights

- Designed a relational data model for users, clients, venue rooms, event bookings, staff assignments, and shift notes.
- Implemented JWT authentication and role-based access for Admin, Manager, Staff, and Demo users.
- Built responsive React workflows for bookings, staff assignments, status updates, dashboard metrics, and shift note capture.
- Added automated backend and frontend tests plus Docker image validation in CI.
- Published a free live demo with seeded demo data and public documentation.
