# Resume Bullets

- Built a full-stack event operations platform using ASP.NET Core, React, PostgreSQL, Docker, and GitHub Actions to manage venue bookings, staff assignments, guest counts, and shift notes.
- Designed REST API endpoints, relational database models, role-based access control, and validation logic for realistic event management workflows.
- Added automated tests, Docker Compose setup, seeded demo data, and CI pipeline to demonstrate production-style development practices.
- Implemented a responsive React dashboard with role-aware navigation, booking filters, status updates, staff assignment workflows, and shift note entry.
