# VenueOps - Full-Stack Event Operations Platform

Built a full-stack web application for managing venue bookings, staff assignments, guest counts, and shift notes using ASP.NET Core, React, PostgreSQL, Docker, and GitHub Actions.

- Designed and implemented REST APIs and SQL-backed workflows for bookings, clients, venues, staff assignments, and event status tracking.
- Built role-based dashboards for admins, managers, and staff to support realistic venue operations.
- Containerized the application with Docker and added GitHub Actions CI to build and test the project automatically.
- Documented setup, architecture, API structure, screenshots, demo accounts, and future improvements in a professional GitHub README.
